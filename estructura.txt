── RUTAS API ──────────────────────────────

Método   Ruta                              Auth  Descripción
───────  ─────────────────────────────────  ────  ──────────────────────────────
POST     /api/auth/register                  —    Registro
POST     /api/auth/login                     —    Login → JWT
GET      /api/auth/me                        ✓    Usuario actual

GET      /api/movies/search?q=&page=         —    Búsqueda de películas
GET      /api/movies/search/multi?q=         —    Películas + personas (TMDB multi)
GET      /api/movies/trending                —    Trending semana/día
GET      /api/movies/popular                 —    Populares
GET      /api/movies/genres                  —    Lista de géneros
GET      /api/movies/discover               —    Filtrado avanzado (géneros, año,
         ?genres=&year_gte=&year_lte=              rating, sort, persona, provider)
         &rating_gte=&sort_by=
         &with_person=&with_watch_providers=
GET      /api/movies/recommendations/me      ✓    Recomendaciones personalizadas
GET      /api/movies/providers?region=       —    Proveedores de streaming
GET      /api/movies/collection/:id          —    Colección/saga
GET      /api/movies/:id                     —    Detalle (credits, videos, keywords)
GET      /api/movies/:id/similar             —    Películas similares
GET      /api/movies/:id/recommendations     —    Recomendaciones TMDB para película

GET      /api/tv/trending                    —    Series trending (semana)
GET      /api/tv/popular                     —    Series populares

GET      /api/people/:id                     —    Persona + filmografía

GET      /api/users/favorites                ✓    Listar favoritos
POST     /api/users/favorites                ✓    Añadir favorito
DELETE   /api/users/favorites                ✓    Eliminar favorito

── MOOD TRANSFER ENGINE ───────────────────

Flujo:
  GAME_CATALOG[n].meta
    └── translateMetaToTMDB(meta)
          ├── theme[] → THEME_TO_GENRES → Set de genre IDs (máx 3)
          ├── mood[]  → sort_by (epic/fun/explosive → popularity.desc)
          └── mood[] + pacing[] → rating_gte (6.0 → 6.5 → 7.0 → 7.5 con prestige)
    └── movieService.discover({ genres, sort_by, rating_gte, page: 1 })
    └── hasta 16 películas en carrusel

Inverse (base preparada, sin UI):
  MOVIE_TO_GAME_HINTS[] — mapea temáticas de película a juegos sugeridos

── NOTAS ARQUITECTURA ─────────────────────

- Dependencia circular api ↔ authStore resuelta con tokenStore.js (módulo plano)
- Double-fetch en StrictMode evitado con useRef(false) guard en efectos de carga
- TV items normalizados en backend con mediaType:'tv' para distinguirlos en frontend
- IntersectionObserver con rootMargin:'400px' para pre-cargar antes de llegar al final
- Rate limit TMDB: cargas escalonadas con setTimeout(fn, i * 300)
- Debounce de filtros: 450ms — evita llamadas durante escritura
- Grid siempre visible (sin modo filtro separado): los filtros cambian contenido, no layout
